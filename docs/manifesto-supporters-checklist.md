## Vernieuwen van deze lijst

1. Pagina-JSON ophalen:  
   `curl -sL "https://fides.community/wp-json/wp/v2/pages/7" -o manifesto-page.json`
2. Script draaien (schrijft JSON naar stdout):  
   `node scripts/fetch-manifesto-supporters.mjs manifesto-page.json > manifesto-supporters-raw.json`
3. Handmatig dit markdown-document bijwerken op basis van de JSON en je visuele check op de live site.
